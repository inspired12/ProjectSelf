# Analyze Questions Skill

Deeply analyze a question set for quality, coverage, and mission alignment.

## Goal

Evaluate question quality and suggest improvements for better knowledge capture.

## Approach

1. **Load and parse questions**
   - Read from JSON or Excel
   - Validate format and structure

2. **Quality analysis**
   - Open-ended vs closed questions
   - Depth level (surface vs deep)
   - Clarity and specificity
   - Potential for rich responses

3. **Coverage analysis**
   - Category distribution
   - Topic gaps
   - Redundancy detection
   - Balance across themes

4. **Mission alignment check**
   - Biology-humanity-technology integration
   - Generation Alpha relevance
   - Systems thinking emphasis
   - Future-oriented perspective

5. **Generate report with**
   - Overall quality score
   - Category distribution chart
   - Top 10 best/worst questions
   - Specific improvement suggestions
   - Gap analysis (missing topics)

## Success Criteria

- [ ] All questions analyzed for quality metrics
- [ ] Category distribution visualized
- [ ] Specific recommendations provided
- [ ] Gap analysis shows missing topic areas
- [ ] Output saved to markdown report
